Films4You has the following requirements for you to implement:
1. Find the total number of actors.
2. Find the top 10 most popular films based on number of rentals.
3. Find all the actors with first name PENELOPE.
4. Find the most popular film in each category based on number of rentals.

They also have other requirements; the system shall:
  i. run using Java 11;
 ii. follow Google's Style guide for Java as enforced by CheckStyle;
iii. have each requirement in its own package and an entry point that implements RequirementInterface;
 iv. be unit tested using JUnit 4 to prove correctness;
  v. use Maven as its build system; and,
 vi. make use of docstring commenting in non-test classes.
